# LaTeX 中文简历模板

一个简洁、紧凑、适合中文场景的 LaTeX 简历模板。

## 文件说明

- `resume_templete.tex`：开源模板

## 模板特点

- 中文友好：基于 `ctex`，支持中文内容排版
- 布局紧凑：页边距和段间距已优化，适合一页简历
- 结构清晰：教育、实习、论文、科研、荣誉等模块开箱即用
